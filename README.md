# Buat root di VPS
- Step 1
```
sudo su
```
- Step 2
```
cd
```
- Step 3
```
apt update && apt install wget -y && wget -qO- -O rootvps.sh https://raw.githubusercontent.com/lesta-2/rootvps/rootvps.sh && bash rootvps.sh
  
```
